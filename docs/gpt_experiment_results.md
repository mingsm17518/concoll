## 分析

1. **置信度分布**：GPT-4o 的置信度普遍很高，大部分样本在 Stage 1 被接受（87.6%）
2. **Stage 2**：th2=0.8 时，92.6% 样本被接受，仅 8 个样本进入 Stage 3
3. **Stage 3**：8/870 样本（0.9%），多 agent 投票结果一致（大部分全票通过）
4. **精度问题**：Accuracy=50.69%，模型存在预测偏向（偏向预测 Vulnerable）
5. **费用**：全数据集约 $7

### 模型预测偏向

- **TP=356, FP=350**：大量将正常代码误判为漏洞
- **TN=85, FN=79**：漏检率相对较低
- 模型倾向于预测 "Vulnerable"，导致 Recall 较高但 Precision 低
